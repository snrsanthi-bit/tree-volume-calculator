# Tree Calculation

林業現場向けの簡易計算ツールです。

## 開発背景

林業をしている友人から  
「社員が材積の計算をうまくできない」「アプリでどんなことができるかわからない」
という話を聞いたことがきっかけで作成しました。

現場では電卓や材積表を使うことも多いですが、
スマートフォンで簡単に計算できるツールがあれば便利だと考え、
簡易的な計算アプリとして作成しています。

## 機能

-材積計算　(直径・高さ)
-地積計算  (三角測量)
-複数三角形による面積合算
-㎡ / ha 表示
